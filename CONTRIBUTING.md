# Contributing Guidelines

Thank you for taking the time to contribute to this project!

## How to contribute

1. Fork this repository.
2. Create a new branch for your change  
   (for example: `feature/my-change` or `bugfix/issue-1`).
3. Make your changes locally and test them.
4. Commit your work with a clear commit message.
5. Push the branch to your fork.
6. Open a Pull Request and describe:
   - what you changed  
   - why the change is needed

## Code of Conduct

By contributing to this project, you agree to follow the rules in `CODE_OF_CONDUCT.md`.
